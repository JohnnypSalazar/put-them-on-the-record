This is the public front door.

The court accepted the payment, accepted the paperwork, and then the record still allegedly treated the person as invisible. That contradiction is the pillar.

- `index.html` is the root $6.50 landing page.
- `six_fifty_record/` is the public-safe record pack.
- `free_legal_packet_builder.html` is the browser-only free packet builder.
- `record.html` is the Put Them On The Record lane.

- `projects.html` is the Scientist Poser / projects doorway.

## Current builder logic

The Free Legal Packet Builder runs in the browser.

It does not submit data.
It does not save data.
It does not require an API.
It generates Markdown the visitor can copy or download.

## Later API path

If submissions or AI generation are added later, the site can move to a host with form handling or serverless functions. The static public front door remains intact.

## Rule

The Court Took My $6.50 is the root.
Projects do not erase the story.
The public record is controlled, readable, and redacted.
The private engine stays offline.

No restart required.
