## Attorney Review Gate

The free packet builder is for documentation support, issue spotting, evidence organization, and attorney preparation. It does not decide legal claims, crimes, damages, deadlines, remedies, jurisdiction, filing strategy, or legal theory. A qualified attorney should make the final call.

## Public-safe rule

No raw private evidence.
No unredacted private identifiers.
No source code.
No private workflows.

The public gets the clean front door.
