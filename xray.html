X-Ray

Make the computer stop hiding the record.

X-Ray is the file-visibility project: folder chaos, evidence piles, weird exports, duplicate documents, lost PDFs, screenshots, receipts, and “where the hell did that file go?” energy.

It is about seeing what is there before trying to explain what it means.

Sees

Folders, files, document piles, local archives, screenshots, PDFs, and messy evidence universes.

Outputs

Readable indexes, file maps, public-safe summaries, missing-item lists, and “next thing to inspect” paths.

Boundary

No raw private evidence goes public by default. X-Ray helps find and organize; publication is a separate decision.
